# APUNTES JS
## Variables Js.
Las variables en Javascript pueden ser de 2 __declaraciones__ principalmente: 
__LET__: Esta declaracion permite a la variable ser transformada.
__CONST__: Esta declaracion permite a la variable __no__ ser transformada.

# POO
En Js, los objetos los creamos a partir directamente del constructor, el constructor inicializa y define los atributos, se puede hacer antes, pero es meramente aclaratorio, documentacional o en utilización de Typescript, pero poco mas, lo mas común es hacerlo en el constructor.
Por otra parte los metodos son bastante parecidos a lo de Java, definimos que se hace dentro con el/los atributo/s con el "this.", indicando la instancia del atributo en el objeto.
